DATAMUSE — Project Overview

Project Objective

DATAMUSE is designed to simplify the process of data cleaning, exploration, and visualization for users. It automates data preprocessing and provides smart chart suggestions to accelerate data analysis workflows.

User Benefits

  • Quickly clean raw datasets without coding
  • Understand data distribution and patterns visually
  • Generate multiple chart suggestions with reasoning
  • Interactive plotting and download of cleaned data
  • Responsive, modern dashboard UI for smooth experience

Supported Data Types

  • Numeric: Integer and Float columns for histograms, scatter plots, box plots, etc.
  • Categorical: Text columns for bar plots, count plots, pie charts, etc.
  • Datetime: Date/Time columns for line plots and trends

Workflow / Pipeline

1 Upload dataset (CSV or Excel)
2 Automatic data cleaning & preprocessing
3 Column type detection (Numeric, Categorical, Datetime)
4 Generate chart suggestions with reason
5 Plot charts interactively
6 Download cleaned dataset

Key Features

  • Upload & preview datasets
  • Automatic cleaning (missing values, duplicates, formatting)
  • Column type detection & intelligent chart suggestions
  • Plot various charts: Histogram, Box, Scatter, Line, Bar, Heatmap
  • Interactive charts embedded in web page
  • Download cleaned dataset
  • Responsive UI with glassmorphism & gradient buttons

Use Cases

  • Data Analysts: Quickly clean and explore client datasets.
  • Data Scientists: Generate insights without repetitive preprocessing.
  • Business Intelligence: Visualize KPIs and trends effortlessly.
  • Students & Learners: Understand data patterns and relationships.
  • Project Demos: Showcase charts and insights interactively.

Technologies Used

Python Flask Pandas NumPy Matplotlib Seaborn TailwindCSS Anime.js HTML5 CSS3 JS

Project File Structure

  • app.py — main Flask application
  • templates/ — HTML templates (upload, dataclean, chart_suggestions, generate_chart, overview)
  • static/ — CSS, JS, charts
  • uploads/ — uploaded datasets & cleaned files
  • functions/ — reusable functions (data cleaning, chart suggestions, plotting)

Animations & UI Effects

  • Glassmorphism cards with hover elevation
  • Gradient buttons & interactive plot buttons
  • Entry animations with Anime.js (fade-in, slide, staggered cards)
  • Scrollable dataset previews with smooth scrollbar
